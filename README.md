### Installation
- Install node.js (V16 or above)
- Clone this repo
- Install everything through `npm install`.<br>
- Make a file `config/config.json`:
```json
{
    "owners": [
        "userID"
    ],
    "prefix": [
        "d!",
        "d."
    ],
    "token": "token",
    "instaAuth": "",
    "channel": {
        "message-delete": "",
        "warn-activity": ""
    }
}
```
### How to use
type `node index.js` in terminal to run the bot
